What it does

Everything you need, nothing you don't

A single, focused tool. Open your browser overlay, use it, close it — without ever leaving the game.

⌨️

Instant Toggle

Press your configured hotkey to show or hide the overlay without touching the mouse. Default is F3.

🔐

Persistent Sessions

WebView2 preserves cookies and auth across every toggle. You're always already logged in.

🪟

Transparent Overlay

The browser floats on top of your game with no background. Adjust opacity and size to taste.

🎮

Focus-Aware

The overlay automatically hides when Star Citizen loses focus so it's never in the way.

🔧

Configurable Hotkey

F3 not working for you? Change the toggle key to any key combination from the Settings menu.

📥

System Tray

Runs quietly in the background. Right-click the tray icon to access Settings or quit at any time.

🚀

Start with Windows

Optional auto-start so SC HUD is always ready when you launch your game. Toggle in Settings.

🛡️

Smart Navigation

An allowed-origins whitelist keeps the overlay within permitted domains. Safe by default.

The process

Up and running in minutes

No installer, no configuration wizard — download, run, press F3.

1

Download & Run

Download schud.exe and run it. No installation required. WebView2 is pre-installed on Windows 11.

2

Configure

Right-click the tray icon → Settings. Set your overlay URL (default: scbridge.app).

3

Launch Your Game

Start Star Citizen and get into the game as normal.

4

Press F3

The overlay appears over your game, already logged in and ready to use.

5

Press F3 Again

Overlay hides instantly. Game focus returns. You never left.

Settings

Configure it your way

All settings are accessible from the system tray. They're stored in %APPDATA%\schud\settings.json.

Setting Default Description
ToggleHotkey F3 Key combination to show and hide the overlay.
GameProcessName StarCitizen.exe Process to monitor for focus detection.
OverlayUrl https://scbridge.app The URL displayed in the overlay browser.
AllowedOrigins *.scbridge.app Domains the embedded browser is permitted to navigate to.
OverlayOpacity 230 (90%) Overlay transparency, 0–255. Lower = more transparent.
HideWhenGameLosesFocus true Auto-hide the overlay when the game is minimised or unfocused.
AutoStartWithWindows false Launch SC HUD automatically at Windows startup.
Good to know

Notes & requirements

WebView2 Runtime required Windows 11 ships with WebView2 pre-installed. Windows 10 users may need to download the Microsoft Edge WebView2 Runtime from the Microsoft website before running SC HUD.
Run as Administrator (recommended) Star Citizen uses Easy Anti-Cheat and runs as an elevated process. Running SC HUD as administrator allows proper focus-return when the overlay is hidden.
EAC compatibility SC HUD does not inject into the game or modify game memory. It is a standalone overlay using a separate WebView2 window positioned on top of the game.
Developers

Run from source

git clone https://github.com/SC-Bridge/SC-HUD
cd SC-HUD
dotnet run --project src/schud

Requires .NET 9.0 SDK and Windows 10 / 11 x64. Build a self-contained executable with dotnet publish -c Release -r win-x64 --self-contained.
